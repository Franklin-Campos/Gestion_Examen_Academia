# Academia Sofia - Plataforma de Exámenes Virtuales

Sistema de exámenes virtuales para academia con:
- Login de alumnos con DNI + PIN
- Exámenes de 20 preguntas con cronómetro
- Corrección automática
- Resultados con semáforo de colores
- Panel de administración completo
- Reportes exportables a CSV

## Tecnologías

- React + Vite
- Tailwind CSS
- Supabase (Auth, PostgreSQL, Edge Functions)
